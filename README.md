# Proyecto Sitio Web

Este proyecto es un ejemplo de página web construida con **HTML5 y CSS3**, siguiendo buenas prácticas de semántica y diseño responsivo.

## Contenido

- **Estructura básica** con etiquetas semánticas (`header`, `nav`, `main`, `section`, `footer`).
- **Formulario de contacto** funcional con integración a Formspree.
- **Estilos básicos** aplicados mediante un archivo CSS externo (`styles.css`).
- **Google Fonts** implementadas correctamente.
- **Diseño responsivo** con Flexbox, Grid.

## Uso

1. Descarga el proyecto y descomprímelo.
2. Abre `index.html` en tu navegador.
3. Para que el formulario funcione, reemplaza `your_form_id` en el atributo `action` por tu ID de Formspree.
